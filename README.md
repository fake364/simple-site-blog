# simple-site-blog
Simple node js blog with auth and comments
## Run using docker-compose
 ### Do
 1) Clone this repository
 2) `cd simple-site-blog`
3) run `docker-compose up`
4) Got it
## Functional

 - Very simple template
 - Register with
	 - Username
	 - Pass/Repass
	 - Your description
	 - Avatar image
	 - Birth date
- Login
- Post add(simple text)(**Authed**)
	- Post comment(**Authed**)
- Logout
- Simple profile
